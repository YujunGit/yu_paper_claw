# Daily Paper Digest — 2026-04-24

## Theme
Today’s worthwhile theme was **explicit structure at the interface, not just inside the model**. The papers worth keeping did not merely claim controllability or generalization. They changed the representation interface so the model could compose conditions, encode scene state once in 3D rather than per-view, or transfer dynamics across robot bodies using explicit morphology.

## Short overview
This was a **selective day**. I found **three papers worth keeping**.

The strongest conceptual pick is **Controllable Image Generation with Composed Parallel Token Prediction**. It is one of the cleaner attempts to make compositional control in discrete generative models mathematically explicit rather than prompt-heuristic.

The strongest direct 3D-generation pick is **Any 3D Scene is Worth 1K Tokens**. Its main contribution is representational: move scene generation into a fixed-length 3D latent rather than pretending long multiview video sampling is the right abstraction.

**Toward Hardware-Agnostic Quadrupedal World Models via Morphology Conditioning** is more adjacent, but it earns inclusion because it attacks embodiment transfer with an explicit conditioning interface instead of vague “generalist world model” rhetoric.

I also checked several newer April world-model papers. Most felt too close to system packaging, benchmark extension, or scale-up of familiar ingredients to justify inclusion today.

## Most relevant to current work
**Any 3D Scene is Worth 1K Tokens** is the closest direct hit for current interests. The key reusable idea is not just “3D is better than 2D,” but that scene generation should live in a representation whose complexity scales with scene structure rather than with the number of rendered views.

## Novelty / framing / baseline impact
- **Compositional generation framing:** CPTP is a good citation against shallow controllability claims. It offers an actual composition mechanism in discrete latent generation, with condition weighting and negation.
- **3D generation framing:** 1K Tokens is useful if you want to argue that multiview/video diffusion is an expedient but misaligned representation for scene-level 3D generation.
- **World-model transfer framing:** Morphology Conditioning is a good adjacent citation for the claim that explicit known structure can be better than implicit inference from history.
- **Novelty threat:** Low. These papers overlap with explicit-structure themes, but none appears to collapse a broader agenda around compositional controllable world modeling.

## One-paragraph takeaway
The good papers today all improved behavior by choosing a better interface between structure and generation. CPTP composes conditional discrete generation rules directly, instead of treating multiple conditions as one more prompt engineering problem. 1K Tokens changes the latent space itself so 3D scene generation is no longer forced through redundant 2D view stacks. Morphology Conditioning does the same kind of cleanup for embodied prediction: robot body parameters are treated as explicit conditioning variables rather than hidden facts the model must guess from motion history.
