# Daily Paper Digest — 2026-03-12

## Theme
Structured intermediate representations for action and generation: papers that make planning, world modeling, or 3D scene composition more explicit instead of hiding everything inside an end-to-end policy.

## Short overview
Today’s strongest paper is **Bilevel Planning with Learned Symbolic Abstractions from Interaction Data**. It is the most directly useful because it tries to make learned symbolic structure operational: symbolic rules are learned from interaction, then explicitly verified against continuous effect models before execution. The broader pattern across today’s picks is that the worthwhile work is not “more agentic” branding, but tighter interfaces between abstract structure and grounded dynamics or geometry.

## Ranked papers

1. **Bilevel Planning with Learned Symbolic Abstractions from Interaction Data**  
   - Verdict: **Highly relevant**  
   - Role: **Directly relevant**  
   - Why it matters: Explicit learned abstractions plus low-level verification is a real neurosymbolic planning mechanism, not just a slogan.

2. **World2Act: Latent Action Post-Training via Skill-Compositional World Models**  
   - Verdict: **Useful**  
   - Role: **Direct utility + methodological inspiration**  
   - Why it matters: The interesting idea is to align actions to world-model latents rather than pixels, and to use skill decomposition to make world-model rollouts usable across varying horizons.

3. **PARSE: Part-Aware Relational Spatial Modeling**  
   - Verdict: **Useful**  
   - Role: **Conceptual inspiration**  
   - Why it matters: It pushes scene structure below the object level into part-to-part relations, which is a good fit for controllable 3D generation and physically grounded spatial reasoning.

## Most relevant to current work
**Bilevel Planning with Learned Symbolic Abstractions from Interaction Data** is the most relevant. It directly targets the neural-plus-symbolic interface, treats abstraction learning as something that must still survive low-level verification, and is useful for thinking about planning, representation, and failure analysis.

## Novelty / framing / baseline impact
- **Framing impact:** Moderate to strong. Today’s better papers argue for explicit intermediate structure rather than opaque end-to-end policies.
- **Related-work impact:** Useful for neurosymbolic planning, world-model-based action generation, and structured 3D scene reasoning.
- **Novelty threat:** Low unless the user’s claim is specifically “we learn symbolic abstractions from interaction and verify them against grounded dynamics” or “we use part-level relational structure for scene generation.”
- **Weak-paper warning:** A number of same-day “agentic” or “VLA” papers looked under-specified or overclaimed from the abstract alone, so I did not include them.

## One-paragraph takeaway
The strongest recurring idea today is that structure only matters if it constrains something real. In the planning paper, symbolic rules are checked against learned continuous dynamics. In World2Act, action learning is moved from pixel imitation toward latent dynamics alignment, with explicit skill decomposition to handle varying horizons. In PARSE, scene relations are grounded in which object parts physically support or contact each other, not just object-level prepositions. That is a much better direction than superficial modularity theater.
