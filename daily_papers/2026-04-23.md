# Daily Paper Digest — 2026-04-23

## Theme
Today’s strongest theme was **using structure to throw away the wrong information**. The worthwhile papers did not merely add modules. They changed the predictive target or planning signal so the model had to care about geometry, temporal reachability, or animator-usable structure.

## Short overview
This was a **selective day**. I found **three papers worth keeping**.

The strongest direct hit is **Occupancy Reward Shaping**. It extracts long-horizon goal geometry from an occupancy world model into a policy-invariant shaping reward, which is a sharper use of world-model structure than the usual “better latent features” story.

**Mask World Model** is also directly relevant. Its central move, predicting future semantic masks rather than RGB, is a clean example of a geometric bottleneck paying rent for control robustness.

**Animator-Centric Skeleton Generation** is more adjacent, but it is the best 3D paper I found today because the semantic tokenization and density control are genuinely workflow-relevant rather than decorative controllability.

I also checked newer robotics/VLA papers from the same window. Most felt too close to system packaging, benchmark pushing, or broad “world knowledge” marketing to justify inclusion.

## Ranked papers

1. **Occupancy Reward Shaping: Improving Credit Assignment for Offline Goal-Conditioned Reinforcement Learning**
   - Verdict: **Must read**
   - Role: **Directly relevant**
   - Why it matters: It turns occupancy-model geometry into a reusable reward signal with a clear theoretical claim, instead of treating world models as opaque feature extractors.

2. **Mask World Model: Predicting What Matters for Robust Robot Policy Learning**
   - Verdict: **Highly relevant**
   - Role: **Directly relevant**
   - Why it matters: It makes an unusually clean representational argument, that control should predict semantic interaction structure rather than photometric futures, and backs it with robustness results.

3. **Animator-Centric Skeleton Generation on Objects with Fine-Grained Details**
   - Verdict: **Useful**
   - Role: **Adjacent but reusable inspiration**
   - Why it matters: The paper treats controllability as a real interface problem by combining semantic decomposition with soft user control over skeletal density.

## Most relevant to current work
**Occupancy Reward Shaping** is the top pick. The transferable idea is that a world model can be useful not only for rollout or latent planning, but also for extracting a global geometry over futures that changes the learning signal itself.

## Novelty / framing / baseline impact
- **World-model framing:** ORS is a strong citation against vague claims that world models are useful because they “encode temporal information.” It shows one concrete way to recover that information and use it.
- **Structured predictive targets:** MWM is useful if you want evidence that better intermediate structure can improve downstream control by discarding nuisance appearance, not by adding more scale.
- **Controllable 3D structure:** Animator-Centric Skeleton Generation is a good adjacent reference for controllability that actually maps to user-editable structure.
- **Novelty threat:** Low to moderate. ORS and MWM overlap with the broader case for explicit structure, but neither seems to subsume a larger agenda around compositional controllable world modeling.

## One-paragraph takeaway
The papers worth keeping today all improve downstream behavior by narrowing the representation to what the task actually needs. ORS converts occupancy-model dynamics into a global shaping reward that better reflects long-horizon reachability. MWM predicts semantic mask evolution rather than pixels, which makes geometry and contact structure the predictive state instead of lighting and texture. Animator-Centric Skeleton Generation makes a parallel point in 3D: semantic grouping and soft density control matter because they create an interface an animator can actually use, not just a prettier output.
