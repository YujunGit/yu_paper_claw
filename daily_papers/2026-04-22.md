# Daily Paper Digest — 2026-04-22

## Theme
Today’s strongest theme was **explicit structure that actually changes the search space**. The worthwhile papers did not just add symbolic or structural language on top, they used structure to stabilize memory, model delayed causal processes, or couple geometry with articulation.

## Short overview
This was a **good but not broad** day. I found **three papers worth keeping**. **Flow Equivariant World Models** is the clearest direct hit: it gives world-model memory a real geometric mechanism rather than a vague persistence claim. **ExoPredicator** is the best abstraction/planning paper today because it treats exogenous processes as first-class causal objects instead of flattening everything into instantaneous actions. **AniGen** is more adjacent, but the co-generation of geometry, skeleton, and skinning is a genuinely useful example of structure paying rent.

I also looked at several large integrated world-system papers from the past week. Most still felt more like capable system assembly than sharp new conceptual leverage, so I did not pad the digest with them.

## Ranked papers

1. **Flow Equivariant World Models: Memory for Partially Observed Dynamic Environments**
   - Verdict: **Must read**
   - Role: **Directly relevant**
   - Why it matters: It gives long-horizon world-model memory a precise symmetry-based mechanism, unifying self-motion and external motion as flows instead of relying on weak generic memory tokens.

2. **ExoPredicator: Learning Abstract Models of Dynamic Worlds for Robot Planning**
   - Verdict: **Highly relevant**
   - Role: **Directly relevant**
   - Why it matters: It is a serious attempt to learn abstract world models that handle delayed exogenous processes, which is a much better framing than pretending all relevant dynamics are instantaneous agent actions.

3. **AniGen: Unified S^3 Fields for Animatable 3D Asset Generation**
   - Verdict: **Useful**
   - Role: **Adjacent but reusable inspiration**
   - Why it matters: The key idea is not “better asset quality” but jointly representing shape, skeleton, and skinning in a shared field so function is generated with geometry rather than bolted on afterward.

## Most relevant to current work
**Flow Equivariant World Models** is the top pick. The transferable lesson is that explicit structure is most valuable when it stabilizes state under transformation, not when it merely adds another memory bank or auxiliary loss.

## Novelty / framing / baseline impact
- **World-model memory:** FloWM is a strong citation if you want to argue that long-horizon memory should respect motion symmetries rather than store view-bound latent snapshots.
- **Abstract planning with temporal structure:** ExoPredicator is a useful reference against action-only symbolic abstractions, especially when external processes evolve concurrently with the agent.
- **Functional 3D generation:** AniGen is a good example of co-generating usable structure instead of treating articulation as brittle post-processing.
- **Novelty threat:** Moderate only at the interface level. FloWM and ExoPredicator overlap with explicit-structure arguments, but neither collapses the broader agenda of compositional, controllable, geometry-aware world modeling.

## One-paragraph takeaway
The interesting papers today all earn their structure claims by narrowing ambiguity in the right place. FloWM constrains memory with flow equivariance so latent state moves coherently across self-motion and external dynamics. ExoPredicator abstracts not only state but also delayed causal processes, which matters if planning must account for things that keep unfolding after the agent acts. AniGen makes a parallel point in 3D generation: if articulation matters, geometry and rigging should be generated in one shared representation instead of handed off to a brittle post-hoc stage.
