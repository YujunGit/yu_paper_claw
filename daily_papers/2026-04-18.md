# Daily Paper Digest — 2026-04-18

## Theme
Today’s strongest theme was **making hidden constraints explicit at the planning interface**. The worthwhile papers were not the flashy large-system ones, but the ones that gave the agent a more structured object to reason over: latent affordances or graph-structured dynamics.

## Short overview
This was a **thin but still useful day**. I found **two papers worth saving**. **ADAPT** is the most directly relevant paper because it forces embodied planners to reason about *unstated affordance constraints* instead of blindly executing instructions. **Learning Ad Hoc Network Dynamics via Graph-Structured World Models** is further from the target domains, but the mechanism is interesting: it builds a per-entity recurrent latent state with cross-entity attention and then uses imagined rollouts for downstream decision-making.

I also checked **HY-World 2.0**, **POMDP-based Object Search with Growing State Space and Hybrid Action Domain**, and **HyperSpace**. HY-World 2.0 still looks more like a broad system integration paper than a tight mechanism paper. The POMDP search paper seems technically serious, but mostly as a specialized planning/solver contribution rather than a reusable conceptual advance for current interests. HyperSpace is a decent infrastructure paper for VSA practitioners, but not strong enough as a daily pick.

## Ranked papers

1. **ADAPT: Benchmarking Commonsense Planning under Unspecified Affordance Constraints**  
   - Verdict: **Useful**  
   - Role: **Directly relevant**  
   - Why it matters: It argues that embodied planning fails when instructions omit implicit object-state preconditions, and adds an explicit affordance-reasoning module rather than hoping a planner infers them implicitly.

2. **Learning Ad Hoc Network Dynamics via Graph-Structured World Models**  
   - Verdict: **Useful**  
   - Role: **Adjacent but useful inspiration**  
   - Why it matters: It treats world-model state as a structured set of interacting entity latents instead of one flat vector, which is a transferable idea for multi-object or multi-agent systems.

## Most relevant to current work
**ADAPT** is the most relevant paper today. The reusable idea is not just “add affordance prediction,” but to treat missing preconditions as a first-class planning variable and insert a dedicated module between perception and action planning.

## Novelty / framing / baseline impact
- **Planning framing:** ADAPT is useful if you want to argue that instruction-following agents need explicit latent precondition inference, not just better action decoding.
- **Representation framing:** G-RSSM is a helpful citation for using entity-structured latent state in world models when interactions matter.
- **Novelty threat:** Low. Neither paper is a direct collision with the repo’s main themes, though ADAPT overlaps with the broader push toward explicit intermediate structure in embodied planning.
- **Baseline impact:** ADAPT is the stronger related-work citation. G-RSSM is better treated as methodological inspiration than a direct baseline.

## One-paragraph takeaway
The better papers today both reject a naive planner interface. ADAPT says instructions under-specify real-world manipulation because affordances and preconditions are often implicit, so planning needs a dedicated layer for inferring what is actually possible now. G-RSSM says dynamics for interacting entities should not be compressed into a flat latent if the downstream task depends on per-entity structure. Different application domains, same useful lesson: when the interface matches the structure of the decision problem, downstream planning gets more robust.
